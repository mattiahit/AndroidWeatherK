# AndroidWeatherK
App to display weather for current location

###### Main features:
* detecting location via GPS
* showing weather for location
* 5 day forecast for location
* light/dark mode

###### Used technology:
* MVVM Architecture
* Kotlin
* Android SDK
* RXKotlin
* Retrofit 2
* Dagger 2
* MOCKK Tests
