# AndroidWeather
Odnowiona aplikacja wyswietlająca prognoze pogody dla danej lokalizacji.

###### Najwazniejsze funkcje aplikacji:
* pobieranie danych pogodowych dot. lokalizacji
* automatyczne wykrycie lokalizacji

###### Użyta technologia:
* KOTLIN
* Android SDK
* MVVM
* Retrofit 2
* Room
* Dagger2
* RxKotlin
